This document explain to user who want to manage beacons data and other additional information like maps, brochure, geo fencing, etc.

The contents are as follows :

  1. Signup Account.

    This section covers the first step on how to get access to the beacon for developer. User will get free demo after signup to cubeacon.

  2. Managing Apps & Apps Setting.

    This section is a manual on how to manage apps, get Client ID, and Client secret to communicate with SDK.

  3. Beacon Analytics.

    This section shows statistic data of beacon access.

  4. Managing Beacons.

    This section is a manual on how to manage beacon information such as UUID, major, minor, color flag, etc.

  5. Storyline.

    This section is a manual guideline on how to manage the notification that will be displayed on mobile phone while passing through the beacon. Notification available on text or image.

  6. Managing Maps.

    This section is a manual guideline on how to manage maps data. It is very simple to use it. just drag and drop the beacon to image map that is uploaded.

  7. Heatmap.

    This section explains about crowd mapping of customers on spots that you have decided.
