This document explain to user who want to manage beacons data and other additional information like maps, brochure, geo fencing, etc.

The contents are as follows :

  1. Signup Account

    This section covers the first step how to get access for developer. By default user will get free demo after signup process.

  2. Sign Up to KiiCloud

    This service use a third party backend as a service called KiiCloud. So developer must sign up kii account first. This section is manual that cover step by step how to create account in Kii Developer.

  3. Integration with KiiCloud

    This section covers the details how to integrate Kii Account and cubeacon developer account.

  4. Managing Apps

    This section is manual how to managing apps get Client ID and Client secret to communicate with SDK.

  5. Managing Beacons

    This section is manual how to managing beacon information such as UUID, major, minor, color flag, etc.

  6. Managing Maps

    This section is manual how to managing maps data. It is very simply to use just drag and drop beacon to image map that uploaded.

  7. Managing Brochure

    This section is manual how to managing brochure that will be displayed on mobile phone when passing through the beacon.
