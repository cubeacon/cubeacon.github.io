Heatmap is one of tools that show to developer about number of traffic users on the area by using micro location map.

To enjoy this feature, please follow steps below :

  1. If you have not set rule id, heatmap will not visible.
  2. Rule ID will follow setting of beacon analytics on this link.
  3. After setup rule id, you will get heatmap point like this.
