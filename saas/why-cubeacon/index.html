With a low price you will get more powerful features and secure data access..

Cubeacon provides easy to configure more than hundred beacons, easy to track point with indoor map and geo fencing for tracking developer group areas like retail shop group area. User can manage smart advertising by simply uploading attractive brochure. For secure cloud storage data, cubeacon using a third party backend as a service called KiiCloud
