If the package that you have still not enough to handle your api request call. Please upgrade to the higher specification. If you do not upgrade it, the request from the customer will not be processed by the system.

For more details, follow the steps below :

  1. Click plans.
  2. Choose the plan you need.
  3. Click upgrade.
  4. Read the agreement first before continuing.
  5. Finish your transaction.
  6. If the transaction is succeed, system will upgrade your plan.
