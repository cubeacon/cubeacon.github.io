Almost Business Intelligence always have the analytics to help make a decision and see the development of the business. You could enjoy that features on Cubeacon Developer.

To enjoy this feature, please follow steps below :

  1. Login to kii developer.
  2. Go to analytics menu
  3. Klik config menu then add new aggregation rules then choose event data.
  4. Activate your aggregation rules.
  5. See analytics on menu metrics.
  6. Go to Cubeacon Developer Site, if you have not set rule id, dashboard will show warning message like this.
  7. Setting rule id based on rule id when crated aggregation rule.
  8. View beacon analytics on Cubeacon Developer.
