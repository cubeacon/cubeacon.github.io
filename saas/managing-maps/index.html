We know that micro location or usually called indoor mapping is very useful. Imagine that we are an owner of many department stores, We want to monitor which department store is in prospect. Cubeacon answers that issue.

For more details, follow the steps below :

  1. Choose maps menu.
  2. Click add button.
  3. Fill your map data.
  4. You can edit map if there is any wrong informations.
  5. Bind Beacons to map by clicking bind button.
  6. Drag beacon list in the right sidebar.
  7. Save binding.
