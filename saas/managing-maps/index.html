We know that micro location or usually called indoor mapping is very useful. Imagine that we are a owner of many department stores. We want to monitor which department store was prospects. Cubeacon answer that issue. Just go to maps menu then monitor beacon point, we will now all about traffic customer.

For more details, follow the steps below :

  1. choose maps menu.
  2. click add button.
  3. fill your map data.
  4. you can edit map if there is any wrong informations.
  5. bind beacons to map with click bind button.
  6. drag beacon list in the right sidebar.
  7. save binding.
