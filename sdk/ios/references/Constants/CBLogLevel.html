CBLogLevel

CBLogLevel enum specifies different levels of logging that could be used to limit or display more messages in logs.

Definition

typedef NS_ENUM(uint8_t, CBLogLevel ) {
   CBLogLevelNone = 0,
   CBLogLevelError = 1,
   CBLogLevelWarning = 2,
   CBLogLevelInfo = 3,
   CBLogLevelDebug = 4,
};
