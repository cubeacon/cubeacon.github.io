Cubeacon SDK for Android is a library to allow interaction with any beacons. The SDK system requirements are Android 4.3 or above and Bluetooth Low Energy.

Cubeacon SDK allows for :

  1. Integrating with Cubeacon SaaS.
  2. Scanning any beacons on a foreground UI or on background as a service.
  3. Showing alert (foreground) or notifications (background) when any beacons entered region (onEnter), exited region (onExit) and nearest detected (onChange).
  4. Showing scenario based on beacons detected.
  5. Sending analytic to Cubeacon SaaS.
